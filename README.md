# dusty-panda-dune

To install dependencies:

```bash
bun install
```

To run in development (auto-reloads on file changes):

```bash
bun run dev
```

To run once (no watcher):

```bash
bun run start
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
